rad.ui

The rad.ui module provides a themeable UI component system for building interfaces. All components inherit from a base class and support per-instance theming.

Theme System

rad.ui uses a JSON-based theme system. Themes define styles for each component type and can be switched globally or per-instance.

Global Theme

// Get the current global theme
var theme = rad.ui.theme;

// Switch global theme by name
rad.ui.settheme('dark');

// Switch to a custom theme object
rad.ui.settheme(myThemeObject);

// Register a new theme
rad.ui.registertheme('custom', myThemeObject);

// Get a theme by name
var dark = rad.ui.gettheme('dark');

Theme Structure

A theme is a nested object with style definitions for each component:

rad.ui.themes.default = {
    "dtype": "px",

    // Base container styles
    "style": {
        "width": 140,
        "height": "auto",
        "margin": 0,
        "fontSize": 10
    },

    // Shared styles (reusable across components)
    "shared": {
        "titlebar": {
            "style": {
                "color": "white",
                "background": "#222",
                "padding": "10px",
                "cursor": "pointer"
            }
        },
        "content": {
            "style": {
                "padding": "10px",
                "background": "#2a2a2a"
            }
        },
        "container": {
            "style": {
                "background": "#2a2a2a",
                "border": "1px solid #444",
                "borderRadius": "5px"
            }
        }
    },

    // Component-specific styles
    "textbox": {
        "type": "text",
        "style": {
            "width": 80,
            "float": "right"
        }
    },

    "button": {
        "style": {
            "float": "right"
        }
    }
    // ... more components
};

Base Class

rad.ui.base is the parent class for all UI components. It provides common functionality for theming and DOM management.

Constructor Options (common to all components)

Option Type Description
id string Element ID for the component
label string Display label text
value any Initial value
callback function Called when value changes
theme object Per-instance theme override

Base Methods

Method Returns Description
getelement() HTMLElement Returns the component's container DOM element
getvalue() any Returns the current value
getguielement() HTMLElement Returns the inner input/control element
gettheme() object Returns the active theme for this instance
settheme(theme) void Set theme by name or object

Components

rad.ui.textbox

A text input field with optional label.

Constructor Options

OptionTypeDefaultDescription
style object {} Container style overrides
style_label object {} Label style overrides
style_textbox object {} Input element style overrides
name string undefined Form field name attribute

Example

var textbox = new rad.ui.textbox({
    id: 'username',
    label: 'Name',
    value: 'default',
    callback: function(component) {
        console.log('Value:', component.getvalue());
    }
});
document.body.appendChild(textbox.getelement());

rad.ui.slider

A draggable slider with numeric input for precise value entry.

Constructor Options

OptionTypeDefaultDescription
settings.clamp boolean false Clamp value to bounds
settings.upper number 1 Upper bound
settings.lower number -1 Lower bound
settings.int boolean false Round to integers
settings.update boolean false Call callback during drag

Methods

MethodDescription
refresh()Update slider position from current value
set_settings(d)Update settings object
set_to_minimum()Set value to lower bound
set_to_maximum()Set value to upper bound

Example

var slider = new rad.ui.slider({
    id: 'opacity',
    label: 'Opacity',
    value: 0.5,
    settings: {
        clamp: true,
        lower: 0,
        upper: 1
    },
    callback: function(component) {
        console.log('Opacity:', component.getvalue());
    }
});
document.body.appendChild(slider.getelement());

Constructor Options

Methods

MethodDescription

expand()Expand if collapsed
collapse()Collapse if expanded
settitle(text)Update titlebar text

Example

    id: 'settings_panel',

Methods

MethodDescription

An absolutely positioned text display panel with auto-sizing and anchor positioning.

Utility Functions

rad.ui.applystyletohtml(element, themeKey, theme)

Apply theme styles to a DOM element. Useful for styling server-generated HTML with the same theme as rad.ui components.

Parameters

ParameterTypeDescription
element HTMLElement The DOM element to style
themeKey string Dot-notation path into theme (e.g., "textbox", "shared.container")
theme object Optional theme object, defaults to rad.ui.theme

Example

// Style a server-generated input
var input = document.getElementById('login_username');
rad.ui.applystyletohtml(input, 'textbox');

// Style a container with shared styles
var container = document.querySelector('.login_form');
rad.ui.applystyletohtml(container, 'shared.container');

// Use a custom theme
rad.ui.applystyletohtml(element, 'button', myCustomTheme);
