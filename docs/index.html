Rad Library Documentation

Rad is a vanilla JavaScript utility framework providing WebGL rendering, UI components, math utilities, and more. Select a module below to view its documentation.

rad.ui

UI component system with theming support. Includes textbox, button, slider, collapsible, list, and more.

rad.chainsaw

WebGL2 renderer with sprite batching, normal mapping, and shadow support.

rad.vector

Vector math utilities for 2D and 3D operations.

rad.matrix

Matrix math for transformations and projections.

rad.io

File I/O and backend communication utilities.
